## Vodafone Kabel Router

### Modelle

|        Model         | DOCSIS | Max. Geschwindigkeit | WiFi |
| :-----------------: | :----: | :------------------: | :--: |
| Vodafone Station    | 3.1    | bis 1 Gbit/s        | WiFi 6 |
| Connect Box         | 3.0    | bis 400 Mbit/s      | WiFi 5 |

|        Model         | DOCSIS | Max. Geschwindigkeit | Status |
| :-----------------: | :----: | :------------------: | :----: |
| EasyBox 904 LTE     | -      | LTE-Router           | Eingestellt |

## Besonderheiten

- **Vodafone Station**: DOCSIS 3.1 und WiFi 6
- **Connect Box**: DOCSIS 3.0 und WiFi 5
- **Technologie**: Betrieb an Vodafone Kabelanschlüssen

## Verfügbarkeit

- **Bezug**: Über Vodafone Kabel erhältlich
- **Zuweisung**: Erfolgt tarifabhängig

## Technologie

- **DOCSIS 3.1**: Für Gigabit-Geschwindigkeiten
- **DOCSIS 3.0**: Für Geschwindigkeiten bis 400 Mbit/s
- **Rückkanal**: Upload über separaten Rückkanal
