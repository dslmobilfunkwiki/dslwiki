## Status der HomeBoxen

- **Nicht mehr verfügbar**: O2 bietet keine eigenen Router mehr an
- **Bestandskunden**: Vorhandene HomeBoxen funktionieren weiterhin
- **Neuverträge**: O2 empfiehlt externe Router (z.B. FritzBox)

## Alternativen

Da O2 keine eigenen Router mehr anbietet:

- **FritzBox**: Vollständig kompatibel mit O2 DSL
- **Andere Router**: Die meisten DSL-Router funktionieren mit O2
- **Provider-unabhängig**: O2 DSL läuft über das Telekom-Netz

## Technische Details

- **DSL über Telekom-Netz**: O2 nutzt Telekom-Infrastruktur (Resale)
- **Standard-PPPoE**: Normale DSL-Einstellungen verwendbar
- **Keine Besonderheiten**: Wie bei anderen DSL-Anbietern
